## 📌 Project Overview
This project demonstrates a basic IoT system using Arduino simulation.

A sensor is used to detect environmental conditions, and based on the sensor input, an LED is automatically turned ON or OFF.

## ⚙️ Components Used
- Arduino
- Sensor (LDR / Temperature / Motion Sensor)
- LED
- Resistors
- Tinkercad (Simulation Platform)

## 🔄 Working Principle
The sensor collects data from the environment and sends it to the Arduino.  
The Arduino processes this data and checks whether it crosses a predefined threshold.  

- If the condition is met → LED turns ON  
- Otherwise → LED remains OFF  

## 🌐 Application
This type of system can be used in:
- Smart street lighting  
- Automatic home lighting  
- Environmental monitoring systems
